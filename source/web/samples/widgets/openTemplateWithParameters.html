Create Dataview from Template with Parameters

Overview

The Dataview API provides the ability to create a new Data View from a Template. After the Data View has been created, you may then launch and embed an entire worksheet in an external HTML page.

Below you will see all of the code necessary to embed a Centrifuge worksheet into a custom application of your own.

Sample Code

<html>
	<head>
		<script type="text/javascript" src="/Centrifuge/javascript/jquery.js"></script>
		<script type="text/javascript" src="/Centrifuge/javascript/centrifuge-widget.js"></script>
		<script type="text/javascript">
			var dv = new DataviewTemplate("<templateUuid>", "Dataview From w/Parameters Template Example", "Centrifuge"); // replace templateUuid with the uuid of the template to be opened
			dv.createDvFromTemplate(function(){$("#set-parameters").removeAttr("disabled");});
			
			function setParameters() {
				dv.setParameters("Phone1", "824", function(){launchDV();});
			}
			
			function launchDV() {
				dv.launch();
			}
		</script>
	</head>
	<body>
		<button id="set-parameters" onClick="setParameters()" disabled="disabled">Click to set Phone1 Parameter to '824'</button>
		<div id='Centrifuge'></div>
	<body>
</html>
