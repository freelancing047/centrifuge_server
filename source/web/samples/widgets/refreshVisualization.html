Refresh Visualization

Overview

The Visualization API provides the ability to perform basic actions on visualizations within a Dataview.

This sample allows you to exert some control over the visualization by being able to tell the Relationship Graph when you would like it to refresh.

Sample Code

<html>
	<head>
		<script type="text/javascript" src="/Centrifuge/javascript/jquery.js"></script>
		<script type="text/javascript" src="/Centrifuge/javascript/centrifuge-widget.js"></script>
		<script type="text/javascript">
			var dv = new Dataview("<dvUuid>", "Centrifuge"); // replace dvUuid with the uuid of the dataview to be opened
			dv.launch();  //embed the flash content into the html page
			var viz = dv.getVisualization("Relationship Graph");  //get the visualization object by name
			function refreshViz() {
				viz.reloadViz();
			}
		</script>
	</head>
	<body>
		<button onClick="refreshViz()">Refresh Relationship Graph</button>
		<div id='Centrifuge'></div>  <!-- container div for flash content -->
	<body>
</html>
