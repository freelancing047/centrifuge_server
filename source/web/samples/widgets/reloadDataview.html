Reload Dataview

Overview

The Dataview API provides the ability to reload the dataview cache after it has already been created.

This may be useful when the source data has changed or you would like to update your original query to include additional information.

This sample shows the basic API calls to refresh the server cache from the client side.

Sample Code

<html>
	<head>
		<script type="text/javascript" src="/Centrifuge/javascript/jquery.js"></script>
		<script type="text/javascript" src="/Centrifuge/javascript/centrifuge-widget.js"></script>
		<script type="text/javascript">
			var dv = new Dataview("<dvUuid>", "Centrifuge"); // replace dvUuid with the uuid of the dataview to be opened
			dv.launch();  //embed the flash content into the html page
			
			function reloadDV() {
				dv.reload(function(){alert('Dataview Has Been Reloaded');});
			}
		</script>
	</head>
	<body>
		<button onClick="reloadDV()">Click to Reload Dataview Cache</button>
		<div id='Centrifuge'></div>  <!-- container div for flash content -->
	<body>
</html>
