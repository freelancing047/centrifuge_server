Create Dataview from Template

Overview

The Dataview API provides the ability to create a new Data View from a Template. After the Data View has been created, you may then launch and embed an entire worksheet in an external HTML page.

Below you will see all of the code necessary to embed a Centrifuge worksheet into a custom application of your own.

Sample Code

<html>
	<head>
		<script type="text/javascript" src="/Centrifuge/javascript/jquery.js"></script>
		<script type="text/javascript" src="/Centrifuge/javascript/centrifuge-widget.js"></script>
		<script type="text/javascript">
			var dv = new DataviewTemplate("<templateUuid>", "Dataview From Template Example", "Centrifuge"); // replace templateUuid with the uuid of the template to be opened
			
			function createDV() {
				dv.createDvFromTemplate(function(){launchDV();});
			}
			
			function launchDV() {
				dv.launch();
			}
		</script>
	</head>
	<body>
		<button onClick="createDV()">Click to Create Dataview from Template</button>
		<div id='Centrifuge'></div>
	<body>
</html>
