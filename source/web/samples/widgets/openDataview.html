Open Dataview

Overview

The Dataview API provides the ability to launch and embed an entire worksheet in an external HTML page.

This sample embeds and loads a Centrifuge worksheet into an html element on the page.

Note: If you have uploaded the appropriate dataview to the server, you should also see a sample below.

Sample Code

<html>
	<head>
		<script type="text/javascript" src="/Centrifuge/javascript/jquery.js"></script>
		<script type="text/javascript" src="/Centrifuge/javascript/centrifuge-widget.js"></script>
		<script type="text/javascript">
			var dv = new Dataview("<dvUuid>", "Centrifuge"); // replace dvUuid with the uuid of the dataview to be opened
			dv.launch();  //embed the flash content into the html page
		</script>
	</head>
	<body>
		<div id='Centrifuge'></div>  <!-- container div for flash content -->
	<body>
</html>
