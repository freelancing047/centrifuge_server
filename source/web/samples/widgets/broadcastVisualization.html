Broadcast Visualization

Overview

The Visualization API provides the ability to perform basic actions on visualizations within a Dataview.

This sample showcases some of the automation features of the API. In this scenario, you may perform an action of the Centrifuge user interface by broadcasting the selection of one visualization to another.

Sample Code

<html>
	<head>
		<script type="text/javascript" src="/Centrifuge/javascript/jquery.js"></script>
		<script type="text/javascript" src="/Centrifuge/javascript/centrifuge-widget.js"></script>
		<script type="text/javascript">
			var dv = new Dataview("<dvUuid>", "Centrifuge"); // replace dvUuid with the uuid of the dataview to be opened
			dv.launch();
			var table = dv.getVisualization("Table");
			var rg = dv.getVisualization("Relationship Graph");
			rg.setVisualizationAttached("true", function(){$("#broadcastViz").removeAttr("disabled");});
			function broadcastViz() {
				table.broadcastViz();
			}
		</script>
	</head>
	<body>
		<ol>
			<li>Make a selection in the 'Table' Visualization</li>
			<li><button id="broadcastViz" onClick="broadcastViz()" disabled="disabled">Broadcast Table Selection</button></li>
		</ol>
		<div id='Centrifuge'></div>  <!-- container div for flash content -->
	<body>
</html>
